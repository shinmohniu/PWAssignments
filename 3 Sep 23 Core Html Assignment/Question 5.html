A desctiption list of Full Stack Web Development tech stack

HTML
HTML is the standard markup language for creating Web pages
CSS
Cascading Style Sheets is a style sheet language used for describing the presentation of a document written in a markup language such as HTML
Css Frameworks
A CSS framework is a library allowing for easier, more standards-compliant web design using the Cascading Style Sheets language.
Version Control System
Version control, also known as source control, is the practice of tracking and managing changes to software code
JavaScript
It is a programming language that is one of the core technologies of the World Wide Web, alongside HTML and CSS.
DSA
It deals with the manner data will be sorted, stored and fetched and also the algorithms which will be used to manage it
ReactJS
React is a free and open-source front-end JavaScript library for building user interfaces based on components
Networking
Computer networking is a process of connecting two more than two computers with the purpose to share data, provide technical support, and to communicate. These networked devices use a system of rules, called communications protocols, to transmit information over physical or wireless technologies.
Database
A database is an organized collection of data stored and accessed electronically through the use of a database management system.
NodeJS & ExpressJS
Node. js is a platform for building the i/o applications which are server-side event-driven and made using JavaScript. Express. js is a framework based on Node. js which is used for building web-application using approaches and principles of Node.
